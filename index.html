01 About

Narrative

Hey — I'm Patrick. I've spent the last decade somewhere between product ownership and getting my hands dirty with infrastructure. Currently I'm at SSC-Services where I look after DevOps platforms, herd cloud migrations, and try to convince development teams that agile isn't just a buzzword.

Lately I've been deep into agentic coding — using Claude Code and OpenCode to build things faster than I ever could solo. I write a lot of backend services and web tools for myself, mostly to solve problems I got annoyed by one too many times. If a task takes me more than twice, it gets automated or turned into a small app.

The stuff I'm most proud of usually happens after hours: a Proxmox homelab that keeps growing, game servers for friends, Hytale mods nobody asked for, custom web dashboards for things that didn't need dashboards, and an EVE Online spreadsheet habit I'm not ready to talk about.

When I'm not staring at a terminal, I'm usually playing guitar. My taste runs the full spectrum from jazz to metal — think Coltrane one evening, Meshuggah the next. Music keeps me sane when the build pipeline doesn't.
